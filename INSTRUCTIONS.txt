WELCOME and THANK YOU.

DreamDMI is a Linux command-line tool for Disease Module Identification in molecular networks, leveraging the top performing methods of the “Disease Module Identification (DMI) DREAM Challenge”.
For more information on the Challenge, Please visit: https://www.synapse.org/modulechallenge

1) GET THE CODE
git pull https://github.com/mattiat/DREAM_DMI_Tool.git

2) INSTALL DOCKER OR SINGULARITY
https://www.docker.com or http://singularity.lbl.gov

3) INSTALL
$ ./install
You will be proposed to test your installation

4) RUN
From any location, run:
$ dream_dmi —help
$ dream_dmi --input=/tmp/input.txt --method=M1 --container=docker

Example input: see README.md
